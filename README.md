## ‚ÑπÔ∏è Overview

This is my NPC generator that I made as a faster and more efficient solution to this problem. It functions by first inputting the amount of NPCs that you want to generate. Then you input the name and gender of the NPC and get the rest of the attributes given to you. It will output the height, hostility, name, gender, profession, and personality of the NPC. While doing this, it will play a scrambling animation that cycles through the random possibilities before giving you the attributes. This creates a unique generation experience. I am Aarush Singh and I made it so that people could quickly generate NPCs that are diverse and unique. Having to make lots of NPCs by hand and have them all be unique can be difficult for game development and this provides an effective solution.
